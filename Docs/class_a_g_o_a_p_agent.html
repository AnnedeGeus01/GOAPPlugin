Detailed Description

GOAP Agent responsible for managing goals, actions, and planning.

This class represents an agent that uses the GOAP (Goal-Oriented Action Planning) system to dynamically plan and execute sequences of actions that achieve its goals. It maintains a knowledge of the world, evaluates goals, and reacts to world state changes.
